# pi-xcode

Pi extension for building, running, debugging, and profiling Xcode projects.

## Install

```bash
pi install /path/to/pi-xcode
```

## Tools

- **xcode_build** — Build an Xcode project or workspace
- **xcode_run** — Build, install, and launch on Simulator
- **xcode_debug** — Attach debugger to a running app
- **xcode_profile** — Profile with Instruments

## Development

```bash
npm install
npm test

# Try locally
pi -e ./src/index.ts
```
